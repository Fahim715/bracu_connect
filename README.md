## Overview

**BRACU Connect** is a comprehensive university companion application built exclusively for **BRAC University** students. By centralizing academic tools, schedules, and administrative information into a single platform, the app is designed to reduce friction in everyday student life — from tracking class schedules and exam dates to monitoring payment status and accessing a digital ID card.

---

## Features

| Category | Features |
|---|---|
| **Academics** | CGPA Calculator, CGPA Analysis, Course Details, All Courses View (time, date, room, section) |
| **Scheduling** | Class & Exam Schedules, BRACU Wingman (deadline tracker) |
| **Campus** | BracU Bus Schedule Checker, BracU Bus Routes |
| **Account & Admin** | Payment Status, Completed Credits Tracker, Digital ID Card, In-App Data Sync, Logout |
| **General** | Real-time Notifications, User-Friendly Interface |

> More features are actively being developed and will be released in upcoming versions.
